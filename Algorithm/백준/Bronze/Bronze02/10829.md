# 이진수 변환

## 1. 문제
- 자연수 N이 주어진다. N을 이진수로 바꿔서 출력하는 프로그램을 작성하시오.

## 2. 입력
- 첫째 줄에 자연수 N이 주어진다. (1 ≤ N ≤ 100,000,000,000,000)

## 3. 출력
- N을 이진수로 바꿔서 출력한다. 이진수는 0으로 시작하면 안 된다.

## 4. 예제 입력
```
53
```

## 5. 예제 출력
```
110101
```

## 6. 코드
```c++
#include <iostream>
#include <string>
#include <vector>
using namespace std;

int main()
{
	vector<char> s;
	long long int n;
	cin >> n;
	

	while (n != 0) {
		int t = (n % 2);
		s.push_back('0' + t);

		n /= 2;
	}

	for (int i = s.size() - 1; i >= 0; i--) cout << s[i];
  return 0;
}
```
