# 별 찍기 - 3

## 1. 문제
- 첫째 줄에는 별 N개, 둘째 줄에는 별 N-1개, ..., N번째 줄에는 별 1개를 찍는 문제

## 2. 입력
- 첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.

## 3. 출력
- 첫째 줄부터 N번째 줄까지 차례대로 별을 출력한다.

## 4. 예제 입력
```
5
```

## 5. 예제 출력
```
*****
****
***
**
*
```

## 6. 코드

```c++
#include<stdio.h>

int main() {
    int n;
    scanf("%d", &n);
    
    for(int i = n; i > 0; i--) {
        for(int j = i; j > 0; j--) {
            printf("*");
        }
        printf("\n");
    }
  
  	return 0;
}
```
