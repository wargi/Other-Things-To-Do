# A+B - 7

## 1. 문제

- 두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

## 2. 입력
- 첫째 줄에 테스트 케이스의 개수 T가 주어진다.
- 각 테스트 케이스는 한 줄로 이루어져 있으며, 각 줄에 A와 B가 주어진다. (0 < A, B < 10)

## 3. 출력

- 각 테스트 케이스마다 "Case #x: "를 출력한 다음, A+B를 출력한다. 테스트 케이스 번호는 1부터 시작한다.

## 4. 예제 입력

```
5
1 1
2 3
3 4
9 8
5 2
```

## 5. 예제 출력
```
Case #1: 2
Case #2: 5
Case #3: 7
Case #4: 17
Case #5: 7
```

## 6. 코드

```c++
#include<stdio.h>

int main() {
    int x, a, b;
    scanf("%d", &x);
    
    for(int i = 0; i < x; i++) {
        scanf("%d %d", &a, &b);
        printf("Case #%d: %d\n", i+1, a+b);
    }
    
    return 0;
}
```
