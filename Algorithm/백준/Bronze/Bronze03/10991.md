# 별 찍기 - 16

## 1. 문제

- 예제를 보고 규칙을 유추한 뒤에 별을 찍어 보세요.

## 2. 입력

- 첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.

## 3. 출력

- 첫째 줄부터 N번째 줄까지 차례대로 별을 출력한다.

## 4. 예제 입력 1

```
1
```

## 5. 예제 출력 1

```
*
```

## 6. 예제 입력 2

```
2
```

## 7. 예제 출력 2

```
 *
* *
```

## 8. 예제 입력 3

```
3
```

## 9. 예제 출력 3

```
  *
 * *
* * *
```

## 10. 예제 입력 4

```
4
```

## 11. 예제 출력 4

```
   *
  * *
 * * *
* * * *
```

## 12. 코드

```c++
#include<stdio.h>

int main() {
    int n;
    scanf("%d", &n);

    for (int i = 0; i < n; i++) {
        for (int j = i+1; j < n; j++)
            printf(" ");
        for (int j = 0; j < i*2+1; j++)
            if(j%2 == 0) printf("*");
            else printf(" ");
        printf("\n");
    }
  
  	return 0;
}
```
