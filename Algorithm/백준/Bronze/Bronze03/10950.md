# A+B - 3

## 1. 문제

- 두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

## 2. 입력

- 첫째 줄에 테스트 케이스의 개수 T가 주어진다.
- 각 테스트 케이스는 한 줄로 이루어져 있으며, 각 줄에 A와 B가 주어진다. (0 < A, B < 10)

## 3. 출력

- 각 테스트 케이스마다 A+B를 출력한다.

## 4. 입력 예시

```
5
1 1
2 3
3 4
9 8
5 2
```

## 5. 출력 예시

```
2
5
7
17
7
```

## 6. 코드

```c++
#include<stdio.h>

int main() {
    int t;
    int a, b;
    scanf("%d", &t);
    
    while(t--) {
        scanf("%d %d", &a, &b);
        printf("%d\n", a+b);
    }
    
    return 0;
}
```
