# 합

## 1. 문제
- n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.

## 2. 입력
- 첫째 줄에 n (1 ≤ n ≤ 10,000)이 주어진다.

## 3. 출력
-  1부터 n까지 합을 출력한다.

## 4. 예제 입력
```
3
```

## 5. 예제 출력
```
6
```

## 6. 코드
```c++
#include<stdio.h>
#include<iostream>
using namespace std;

int main() {
    int n, sum = 0;
    scanf("%d", &n);
    
    for(int i = 1; i <= n; i++)
        sum += i;
    
    printf("%d\n", sum);
    
    return 0;
}
```
