# AxB

## 1. 문제
- 두 정수 A와 B를 입력받은 다음, A×B를 출력하는 프로그램을 작성하시오.

## 2. 입력
- 첫째 줄에 A와 B가 주어진다. (0 < A, B < 10)

## 3. 출력
- 첫째 줄에 A×B를 출력한다.

## 4. 예제 입력
```
1 2
```

## 5. 예제 출력
```
2
```

## 6. 예제 입력

```
3 4
```

## 7. 예제 출력

```
12
```

## 8. 코드

```c++
#include <iostream>
using namespace std;


int main()
{
    int n, m;
    cin >> n >> m;

    cout << n * m;
  
  	return 0;
}
```
